JavaScript
JavaScript ("JS" для краткости) - это полноценный динамический язык программирования, который применяется к HTML документу, и может обеспечить динамическую интерактивность на веб-сайтах.
Python
Python – универсальный ЯП высокого уровня, его преимущества: высокая производительность и хорошо читаемый код. Синтаксис Питона максимально облегчен, что позволяет выучить его за сравнительно короткое время.
C++
C++ широко используется для разработки ПО, высокопроизводительных серверов, а также развлекательных приложений, являясь одним из самых популярных языков программирования.
Swift
Swift - язык программирования от Apple с открытым исходным кодом. Предназначен для разработки приложений для iOS и macOS, реже используется в других проектах. Язык появился всего в 2014 году как альтернатива Objective-C.
Kotlin
Kotlin - язык программирования, созданный в компании JetBrains. Его разработали в 2011 году на замену Java, который в компании считали слишком многословным. Новый язык получился на 40% компактнее предшественника.
Java
Java - современный объектно-ориентированный язык программирования. Программа, написанная на Java, способна выполняться практически на любом компьютере.
